# 객체지향 기말 프로젝트
클래스 정의, 상속 등 객체지향 프로그래밍 기능을 이용한 게임 개발 코드

## 객체지향 프로그래밍을 이용한 게임

```
* 게임에서 캐릭터, 아이템, 적 캐릭터 등의 다양한 요소들을 객체로 표현할 예정 
* 슈팅 게임: 플레이어가 적을 공격하고 피하는 게임. 
* 객체지향 프로그래밍을 활용하여 플레이어, 총알, 적 등을 클래스로 구현하고 충돌 감지, 총알 발사, 적의 이동 패턴 등을 개발할 예정
```

```
1. Player 클래스:
    플레이어 위치 (x, y) 및 초기 위치 설정
    이동 메서드: 사용자 입력에 따라 플레이어의 위치를 변경
    발사 메서드: 총알을 생성하고 총알의 초기 위치를 플레이어 위치로 설정

2. Bullet 클래스:
    총알 위치 (x, y) 및 초기 위치 설정
    이동 메서드: 총알이 화면 위로 이동하도록 위치 변경

3. Enemy 클래스:
    적 위치 (x, y) 및 초기 위치 설정
    이동 메서드: 적이 화면 아래로 이동하도록 위치 변경

4. Game 클래스:
    초기화 메서드: 게임 시작 시 필요한 객체 생성 및 초기 설정
    게임 루프: 사용자 입력을 받고, 플레이어 이동, 총알 발사, 적 이동 등 게임 상태 업데이트
    점수 측정: 점수를 화면에 나타냄
    충돌 감지: 플레이어와 적, 총알과 적이 충돌했는지 확인
    게임 종료: 충돌이 감지되면 게임을 종료
```

### 피드백
```
* 레벨 업, 아이템 획득 등의 기능을 추가로 구현 할 수 있음
* 게임 점수 기록 및 사용자 조회를 위해 데이터베이스 개념을 코드에 적용할 수 있음
```

### 코드 틀
```
* Python: 게임의 주요 개발 언어로 사용되었음
* Pygame 라이브러리: 게임 창을 만들고 그래픽 요소를 그리고 사용자 입력을 처리하는 등의 작업을 수행함
* SQLite: 플레이어의 점수를 저장하고 관리함
```
